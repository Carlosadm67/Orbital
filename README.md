### System Requirements

Make sure your computer meets these needs:

- Windows 10 or later
- At least 4 GB of RAM
- 500 MB free storage for the app
- Internet connection to fetch real-time data
- Screen resolution of 1280 x 720 or higher

### What You’ll See in Orbital

- Live telemetry graphs showing satellite data
- Maps with satellite positions

## 🖥 Using Orbital Basics

This section helps you use the main parts of the app.

### Interface Overview

- **Telemetry Dashboard:** Shows live numbers and history of satellite signals.
- **Map View:** Displays the current locations of satellites orbiting Earth.
- **Alerts:** Lists important updates or warnings from the systems.
- **Settings:** Allows you to customize your view and update how often data refreshes.

### Navigate the App

- Use the menu on the left to switch between dashboards.
- Click on graphs or map points to get detailed info.
- Hover or tap elements for tooltips explaining data values.

### Customize Settings

Go to the Settings tab to:

- Change the update speed (default is every 5 seconds).
- Enable or disable alerts.
- Switch between light or dark mode.
- Set units for distance and time.

## 🔧 Troubleshooting and Tips

### If Orbital Does Not Open

- Check if Windows blocked the app (look in the security center).
- Make sure your computer meets the system requirements.
- Restart your PC and try running Orbital again.

### Data Does Not Load

- Verify your internet connection is active.
- Ensure no firewall or antivirus blocks Orbital.
- Check if the version is the latest on the releases page.

### Improving Performance

- Close other programs if Orbital feels slow.
- Set a slower refresh rate in Settings if you have limited internet.

### Update Orbital

When a new release is available, repeat the download and install steps. It is safe to install over your current version.
